# Employee Management System (Full Stack)

A Full Stack Employee Management System built using **Spring Boot** and **React (Vite)**.  
This application allows users to perform CRUD operations on employee data with a modern UI dashboard and charts.

---

## 🚀 Technologies Used

### Frontend
- React (Vite)
- Bootstrap
- Axios
- React Router
- Chart.js
- React Toastify
- React Icons

### Backend
- Spring Boot
- Spring Data JPA
- REST API
- Maven

### Database
- MySQL 

### Tools
- Git
- GitHub
- VS Code
- Eclipse

---

## 📌 Features

- Add Employee
- Update Employee
- Delete Employee
- View All Employees
- REST API Integration
- Full CRUD Operations
